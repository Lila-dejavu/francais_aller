# ⌨️ 法文虛擬鍵盤使用指南

## ✨ 功能介紹

在所有需要輸入法文的題目中，系統會自動顯示**法文虛擬鍵盤**，幫助你輕鬆輸入特殊字母！

---

## 🎯 什麼時候會出現？

虛擬鍵盤會在以下情況自動顯示：

### 1️⃣ 填空題 (type: "input")
- 輸入簡單的法文單字或短語
- 例如：「請輸入法文的『請』」

### 2️⃣ 寫作題 (type: "writing")
- 需要輸入較長的法文作文
- 例如：生日邀請函、搬家通知、論述文等
- 輸入框會自動變大（150px 高度）

---

## 🎮 如何使用

### 方法 1：點擊虛擬鍵盤 ✨（推薦）
1. 在輸入框中輸入文字
2. 需要特殊字母時，點擊虛擬鍵盤上的按鈕
3. 字母會自動插入到**游標位置**
4. 繼續輸入其他內容

### 方法 2：鍵盤快捷鍵
如果你的電腦鍵盤支援，也可以使用：

#### Windows
- `Alt + 數字鍵` 方式（需要數字小鍵盤）
- `Alt + 0233` = é
- `Alt + 0232` = è
- `Alt + 0224` = à
- `Alt + 0231` = ç

#### Mac
- `Option + e, 然後按 e` = é
- `Option + \`, 然後按 e` = è
- `Option + \`, 然後按 a` = à
- `Option + c` = ç

## 💡 使用範例

### 範例 1：輸入單字
題目：「請輸入『咖啡』的法文」

```
1. 先輸入 "caf"
2. 點擊虛擬鍵盤的 "é" 按鈕
3. 完成：café ✅
```

### 範例 2：寫作文
題目：寫生日邀請函（40字+）

```
1. 輸入：Salut Sophie! C'est mon anniversaire samedi. 
2. 需要 à：Je fais une f
3. 點擊 "ê" 按鈕
4. 繼續輸入：te chez moi 
5. 點擊 "à" 按鈕
6. 完成：à 19h. Tu viens? ✅
```

## 🔍 技術細節

### 游標位置插入
虛擬鍵盤會**智能插入**字元到游標位置：

```javascript
// 例如：輸入框內容是 "cafe"，游標在 f 和 e 之間
"caf|e"

// 點擊 é 按鈕後
"café|e"

// 繼續編輯
"café"
```

### 視覺反饋
每次點擊按鈕都有微動畫：
- 按下時：`scale(0.95)`
- 釋放後：恢復原狀
- 持續時間：100ms

## 💬 常見問題

### Q: 虛擬鍵盤什麼時候顯示？
A: 所有填空題和寫作題都會自動顯示。

### Q: 可以關閉虛擬鍵盤嗎？
A: 不需要關閉，它不會干擾正常輸入。如果你會用快捷鍵，可以直接打字。

### Q: 大寫字母怎麼打？
A: 虛擬鍵盤提供常用大寫（À, É, È, Ç），其他可用 Shift 鍵。

### Q: 手機上好用嗎？
A: 是的！已針對手機優化排版和按鈕大小。

### Q: 按錯了怎麼辦？
A: 用退格鍵（Backspace）刪除即可。

## ✨ 設計理念

虛擬鍵盤的設計遵循以下原則：

- **無縫集成**：不干擾正常輸入流程

- **美觀實用**：漸層色彩，現代化設計
