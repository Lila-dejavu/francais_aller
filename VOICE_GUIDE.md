# 🔊 語音功能使用說明

## 功能概述

365法文日記現在支援完整的語音朗讀功能！使用 Web Speech API 來朗讀法文單字、句型和答案，幫助你學習正確的法文發音。

## 🎯 主要功能

### 1️⃣ 自動播放
- **開啟狀態**：每當顯示新問題或正確答案時，系統會自動朗讀法文內容
- **關閉狀態**：需要手動點擊🔊按鈕才會播放
- **切換方式**：點擊遊戲介面頂部的「🔊 自動播放：開/關」按鈕

### 2️⃣ 語速控制
- 拖動滑桿調整語速（0.5x - 1.5x）
- 預設語速：0.9x（較慢，更清楚）
- 建議初學者使用 0.7x - 0.9x
- 進階學習者可使用 1.0x - 1.2x

### 3️⃣ 手動播放按鈕

#### 在選擇題中
- 每個法文選項旁都有🔊按鈕
- 點擊即可聽該選項的發音
- 不會觸發答案選擇

#### 在反饋中
- 正確答案旁有「🔊 聽發音」按鈕
- 學習單字時可重複播放

## 📱 使用場景

### 場景1：學習新單字
1. 顯示問題時，系統自動朗讀法文
2. 聽發音，理解正確讀法
3. 選擇或輸入答案
4. 答對後，再聽一次正確發音

### 場景2：練習發音
1. 關閉自動播放
2. 看到法文單字，先自己嘗試發音
3. 點擊🔊按鈕，聽正確發音
4. 重複練習直到掌握

### 場景3：複習單字
1. 調慢語速到 0.7x
2. 仔細聽每個音節
3. 跟著一起念
4. 逐漸提升語速

## 💡 學習技巧

### 初學者（第1-100天）
1. **開啟自動播放**：讓系統自動教你發音
2. **語速設為 0.8x**：聽清楚每個音節
3. **重複播放**：每個單字至少聽3次
4. **跟讀練習**：聽完後立刻跟著念

### 中級學習者（第101-200天）
1. **間歇自動播放**：某些單字手動播放
2. **語速設為 0.9x - 1.0x**：接近正常速度
3. **盲聽練習**：先不看選項，只聽發音
4. **句子練習**：整個句子一起聽

### 高級學習者（第201-365天）
1. **關閉自動播放**：自主控制學習節奏
2. **語速設為 1.0x - 1.2x**：正常到快速
3. **聽力測試**：只聽發音就寫出單字
4. **對話模擬**：聽完後自己造句

### 語音語言
系統會自動選擇法文語音：
1. 優先選擇 `fr-FR`（法國法文）
2. 其次選擇包含 `fr` 的語音
3. 如找不到法文語音，使用預設語音

## 🎵 發音特點

### 法文特殊發音
- **鼻音**：an, on, in, un
- **連音**：vous_avez
- **省音**：l'école（不是 le école）
- **重音**：最後音節

### 常見單字發音
```
Bonjour     → 「棒朱」
Merci       → 「梅西」
S'il vous plaît → 「西伏普雷」
Au revoir   → 「喔喝瓦」
Comment     → 「口芒」
```

## ❓ 常見問題

### Q1: 沒有聲音？
**A**: 
1. 檢查電腦音量
2. 確認瀏覽器允許播放聲音
3. 嘗試重新整理頁面
4. 檢查是否開啟自動播放

### Q2: 發音聽起來不像法文？
**A**: 
1. 檢查瀏覽器是否有法文語音包
2. Chrome 通常有最好的法文支援
3. Mac Safari 使用 Siri 法文語音，品質較好

### Q3: 語音播放延遲？
**A**: 
1. 關閉自動播放，改用手動
2. 等待語音引擎初始化完成
3. 重新整理頁面

### Q4: 能否下載語音？
**A**: 
- Web Speech API 不支援下載
- 但你可以重複播放練習

### Q5: 可以播放整個句子嗎？
**A**: 
- 可以！系統會播放包含在 `frenchText` 中的所有內容
- 長句子會完整朗讀

## 🚀 進階用法

### 1. 調整音調（開發者選項）
在瀏覽器控制台輸入：
```javascript
window.game.voiceManager.pitch = 1.2; // 提高音調
window.game.voiceManager.speak("Bonjour");
```

### 2. 查看可用語音
```javascript
window.speechSynthesis.getVoices().forEach(v => 
  console.log(v.name, v.lang)
);
```

### 3. 強制使用特定語音
```javascript
const voices = window.speechSynthesis.getVoices();
const frenchVoice = voices.find(v => v.name.includes("Thomas")); // Mac法文男聲
window.game.voiceManager.frenchVoice = frenchVoice;
```
