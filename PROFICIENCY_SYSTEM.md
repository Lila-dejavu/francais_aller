# 📊 熟練度系統使用說明

## 🎯 功能簡介

自訂題庫現在加入了**間隔重複學習系統**（Spaced Repetition System），會根據你對每個句子的熟練度智能調整出題頻率。

## ✨ 主要特色

### 1. 每次練習30題
- 自訂題庫每次練習限制最多 **30 題**
- 如果總題庫少於30題，則全部出題
- 如果總題庫超過30題，系統會智能選擇最需要練習的題目

### 2. 熟練度評分
答題後會看到兩個按鈕：
- **✅ 熟練**：表示你已經掌握這個句子
- **❌ 不熟練**：表示需要更多練習

### 3. 智能出題
系統會根據你的選擇調整每個句子的熟練度等級（0-5級）：
- **熟練度 0-1**：需要加強（出現頻率最高）
- **熟練度 2-3**：正在學習（中等頻率）
- **熟練度 4-5**：已經掌握（出現頻率較低）

## 🔄 使用流程

1. **開始練習**
   - 點擊「必學句型」開始自訂題庫練習
   - 系統會根據熟練度選出30題

2. **答題**
   - 像平常一樣回答問題
   - 查看正確答案和解釋

3. **評估熟練度**
   - 看到反饋後，會出現兩個按鈕
   - 根據你的真實感覺選擇「熟練」或「不熟練」

4. **繼續下一題**
   - 選擇熟練度後，「下一題」按鈕才會出現
   - 點擊繼續下一題

5. **完成練習**
   - 完成30題後查看成績
   - 下次練習時，系統會根據上次的熟練度記錄調整題目

## 📈 熟練度等級說明

| 等級 | 狀態 | 出現權重 | 說明 |
|------|------|----------|------|
| 新題目 | 🆕 未學習 | 10（最高）| 從未練習過的句子 |
| 0-1級 | ❌ 需加強 | 8-10 | 標記為不熟練的句子 |
| 2-3級 | 📚 學習中 | 4-7 | 正在學習中 |
| 4-5級 | ✅ 已掌握 | 1-3（最低）| 多次標記為熟練 |

## 💡 使用建議

### 誠實評估
請根據你的**真實感受**選擇：
- 如果你能在1-2秒內想出正確答案 → 選「熟練」
- 如果需要思考很久或經常忘記 → 選「不熟練」

### 不要急於標記為熟練
即使答對了，如果不是非常確定，建議先選「不熟練」，讓系統多次出題加深記憶。

### 定期複習
- 建議每天練習一次自訂題庫
- 系統會自動調整，不會一直出相同的題目
- 已掌握的句子會逐漸減少出現，讓你專注在需要加強的部分

## 🗂️ 資料儲存

熟練度資料儲存在瀏覽器的 localStorage 中：
- **鍵名**：`frenchDiary_proficiency`
- **格式**：JSON 物件
- **內容**：每個句子的熟練度等級、複習次數、最後複習時間

## 🔧 進階功能

### 查看熟練度統計
每次選題時，控制台（F12）會顯示：
```
📊 熟練度分布: 
  new: 5      # 新題目
  low: 10     # 低熟練度
  medium: 8   # 中等熟練度
  high: 7     # 高熟練度
```

### 重置熟練度資料
如果想重新開始，可以在控制台執行：
```javascript
localStorage.removeItem('frenchDiary_proficiency');
location.reload();
```

## 📊 選題演算法

系統使用**加權隨機選擇**：
1. 計算每個題目的權重（熟練度越低，權重越高）
2. 根據權重進行隨機抽選
3. 確保新題目和不熟練的題目有更高機會被選中

公式：`權重 = max(1, 10 - 熟練度等級 × 1.8)`

## ❓ 常見問題

**Q: 如果我想重新練習已經掌握的句子怎麼辦？**
A: 已掌握的句子仍會出現，只是頻率較低。如果想增加練習，可以在答題後選擇「不熟練」。

**Q: 熟練度會自動降低嗎？**
A: 目前不會。如果長時間沒練習某個句子，下次遇到時請誠實評估。

**Q: 可以跳過熟練度選擇嗎？**
A: 目前設計為必選，這樣才能準確追蹤你的學習進度。

**Q: 熟練度資料會消失嗎？**
A: 資料儲存在瀏覽器中，除非清除瀏覽器資料或更換裝置，否則會一直保留。

## 🎓 學習建議

1. **每日練習**：建議每天至少練習一次（30題約需10-15分鐘）
2. **誠實評估**：不要因為答對就選「熟練」，要根據真實掌握程度
3. **重複是關鍵**：不要怕題目重複，重複才能加深記憶
4. **搭配朗讀**：利用🔊按鈕多聽發音，加強聽力和口說
5. **持之以恆**：間隔重複學習需要時間，堅持練習效果最好

---

💡 **小提示**：這個系統的目標是讓你把時間花在最需要練習的地方，而不是一直重複已經會的內容。相信系統的演算法，持續練習，你會發現進步越來越快！
