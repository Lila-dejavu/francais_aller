# 🔊 語音功能快速入門

## ✨ 新功能亮點

365法文日記現在支援完整的**法文語音朗讀**功能！

### 主要特色
- 🎙️ **真人級語音**：使用瀏覽器內建的法文語音引擎
- 🔄 **自動播放**：顯示問題時自動朗讀法文
- 🎚️ **語速控制**：0.5x - 1.5x 自由調整
- 🔘 **手動播放**：每個法文單字/選項都有專屬播放按鈕
- 🔁 **無限重複**：想聽幾次就聽幾次

## 🚀 快速開始

### 1. 開啟遊戲
打開 `index.html`，在遊戲畫面頂部你會看到：

```
┌────────────────────────────────────┐
│ 🔊 自動播放：開  語速：[==○==] 0.9x │
└────────────────────────────────────┘
```

### 2. 自動播放（推薦初學者）
- **預設開啟**：每次顯示問題或答案時自動朗讀
- **點擊按鈕**：可切換開關狀態
- **藍色 = 開啟** / 灰色 = 關閉

### 3. 手動播放（進階學習）
在選擇題選項中：
```
[Bonjour 🔊]  ← 點擊🔊聽發音
```

在答案反饋中：
```
📚 學到了：Bonjour - 你好 [🔊 聽發音]
```

### 4. 調整語速
- 向左拖動 = 變慢（0.5x - 0.9x）推薦初學者
- 向右拖動 = 變快（1.0x - 1.5x）推薦進階者

## 🎯 學習技巧

### 技巧1：影子跟讀（Shadowing）
1. 播放法文發音
2. 立即跟著重複
3. 模仿語調和節奏
4. 逐漸提升語速

### 技巧2：盲聽測試
1. 關閉自動播放
2. 不看文字，只聽發音
3. 嘗試寫下聽到的單字
4. 對照正確答案

### 技巧3：對比練習
1. 自己先念一遍
2. 播放標準發音
3. 對比差異
4. 重複練習直到相似

### 技巧4：情境模擬
1. 聽完整個句子
2. 想像在法國的場景
3. 自己造句練習
4. 用語音驗證發音

## ❓ 常見問題

**Q: 沒有聲音？**
- 檢查電腦音量
- 確認瀏覽器允許播放聲音
- 重新整理頁面

**Q: 發音不像法文？**
- 使用 Chrome 或 Edge（法文語音支援最好）
- Mac 用戶使用 Safari（Siri 法文語音品質高）

**Q: 語音播放很慢？**
- 這是故意的！初學者建議慢速播放
- 可以調整語速滑桿到 1.0x

**Q: 可以下載語音嗎？**
- Web Speech API 不支援下載
- 但可以無限次重複播放
