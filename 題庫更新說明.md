## 🎯 使用流程

### 方法1：使用 Excel 轉換器（推薦）

1. 打開 `excel_converter.html`
2. 選擇格式（筆記本/Tab/豎線/逗號）
3. 貼上句子
4. 選擇分類
5. 點擊「🔄 轉換成題目」
6. 點擊「📋 自動複製」（只會複製程式碼部分）
7. 打開對應的分類檔案（如 `colloquial.js`）
8. 找到 `// === 新增的 口語俚語 題目 ===` 這一行
9. 在這行**下方**按 `Ctrl+V` 貼上
10. 存檔完成！

### 方法2：手動編輯

1. 打開任一分類檔案
2. 找到 `// === 新增的 XXX 題目 ===` 這一行
3. 參考現有格式，在下方加入新題目
4. **重要：**
   - 如果是第一題，題目前面**不要**加逗號
   - 如果前面已有題目，題目前面**要**加逗號
5. 存檔完成！

## 📝 範例

### 正確的新增方式

```javascript
const colloquial = [
    // === 新增的 口語俚語 題目 ===
    // 在這裡貼上新題目，記得上一題後面要加逗號！
    {
        type: "multiple",

        answer: "Bonjour",

        vocabulary: "bonjour - 你好"

    {  // ← 第二題前面有逗號
        type: "multiple",
        question: "「謝謝」的法文是?",
        options: ["Merci", "Bonjour", "Au revoir", "Oui"],
        answer: "Merci",
        frenchText: "Merci",
        explanation: "Merci 是「謝謝」的法文。",
        vocabulary: "merci - 謝謝"
    }
    
    // === 口語俚語 題目結束 ===

```

## 🔧 常見問題

### Q: 為什麼我貼上後題目沒有出現？
A: 請強制重新整理頁面（Ctrl + Shift + R 或 Ctrl + F5）

### Q: 出現語法錯誤怎麼辦？
A: 檢查：
   1. 最後一題的 `}` 後面**不要**加逗號
   2. 中間的題目 `}` 後面**要**加逗號
   3. 所有的引號、括號都要成對

### Q: Excel 轉換器複製時包含說明文字？
A: 新版已修復！點「📋 自動複製」只會複製程式碼部分

### Q: 如何確認題目已正確載入？
A: 按 F12 開啟開發者工具，在 Console 輸入：
   ```javascript
   window.customQuestions.length  // 查看總題數
   window.colloquial.length       // 查看特定分類題數
   ```
